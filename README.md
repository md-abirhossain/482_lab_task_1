# Md. Abir Hossain — Personal Portfolio

A simple, responsive personal portfolio website built with plain HTML and CSS.
Made by Md. Abir Hossain, CSE student at North South University, Dhaka.

## Pages
- **index.html** — Home page with header, navbar, hero section, intro cards and footer.
- **about.html** — About me page with bio, skills and a journey timeline.

## Features
- Consistent color scheme and modern Google Fonts (Poppins + Playfair Display).
- Sticky navbar with hover underline effect and highlighted active page.
- Hero section, styled footer and cards.
- Layouts built with CSS Flexbox and Grid.
- Hover/transition effects on buttons, cards and links.
- Fully responsive for mobile, tablet and desktop (with a mobile hamburger menu).

## How to view
Just open `index.html` in any web browser. No build step required.
